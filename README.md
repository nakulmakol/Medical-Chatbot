This is an AI medical chatbot built using Flask, LangChain, FAISS, HuggingFace embeddings, and Groq LLMs. I used RAG to retrieve information from a medical textbook before generating answers. The project also integrates medical APIs through MCP tools like PubMed and FDA APIs. FAISS performs semantic similarity search, and the final response is generated using the Groq-hosted Llama 3.3 model.
